# Safzzo SaaS Website

**Enterprise Software Solutions — Salem, Tamil Nadu, India**

A full production-ready Next.js 14 + TypeScript + Tailwind CSS website with:
- 🎨 3D animations, glassmorphism, neon effects, particle canvas
- 📱 Fully responsive (mobile-first)
- 🔍 Full SEO: metadata, OpenGraph, Twitter cards, JSON-LD structured data, sitemap, robots.txt
- 🛒 Products page with 3-step checkout modal
- ⚡ Custom animated cursor, typewriter hero, floating 3D orb
- 🇮🇳 India-first: GST, Razorpay, Tamil/Hindi mentions

## SEO Checklist

- [x] Title & meta description on every page
- [x] OpenGraph + Twitter Card images
- [x] JSON-LD: SoftwareApplication + LocalBusiness schemas
- [x] Canonical URLs
- [x] Sitemap.xml
- [x] Robots.txt
- [x] Semantic HTML (h1, h2, section, nav, footer)
- [x] Alt text ready for images
- [x] Mobile-friendly (Google CWV optimised)

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: CSS keyframes + JS canvas
- **Icons**: Lucide React
- **Fonts**: Clash Display + DM Sans (Google Fonts)
